Welcome to spoti.py
===================
spoti.py is an async python wrapper for the Spotify Web API.

Installation
------------
From PyPI

.. tab:: Linux

    .. code:: bash

        python3 -m pip install -U spoti.py

.. tab:: Windows

    .. code:: bash

        py -3 -m pip install -U spoti.py

From GitHub:

.. tab:: Linux

    .. code:: bash

        git clone https://github.com/Axelware/spoti.py
        cd spotipy
        python3 -m pip install .

.. tab:: Windows

    .. code:: bash

        git clone https://github.com/Axelware/spoti.py
        cd spotipy
        py -3 -m pip install .
